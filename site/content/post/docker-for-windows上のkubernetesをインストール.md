久々のKubernetes
===

新しいノートPCになってから、久しぶりにDockerをインストール。以前のマシンはなぜかHyper-Vが立ち上がらずに、Docker自体が使えなくなってしまっていたのでした(VirtualBoxとの切り替えで何かが起きたかもしれないのだけれど追い切れず)

**kube proxy** して管理用のWeb UIが見られる所までは確認。  
あとはhelm入れてGitLabを入れればOKなはず。

Helmインストール
---
Chocolateyにありました

```
choco install helm
```
